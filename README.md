## Prerequisites
* [node](https://nodejs.org/) in PATH
* [Ruby](https://www.ruby-lang.org/)

## Building
```bash
npm install
grunt
```

`node_modules` is ignored in this repository for reasons that should be obvious if you understand node.

Output can be found in the `dist/` directory.
